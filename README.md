## My vim folder

# Installation

0. Clone repo:
```
$ git clone https://github.com/gaynetdinov/vimfiles2.git ~/.vim
```

1. Set up `Vundle`:
```
$ git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
```

2. Create `~/.vimrc` file and put `source ~/.vim/vimrc` there.

3. Launch `vim`, run `:BundleInstall`
